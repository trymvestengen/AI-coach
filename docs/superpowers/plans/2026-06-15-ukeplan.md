# Ukeplan (del 3) — implementasjonsplan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`).

**Goal:** La maler planlegges på ukedager (`scheduled_days`), settes via ⋯-menyen, og vises i en «Denne uka»-widget på Hjem som åpner en popup med ukeplanen.

**Architecture:** `scheduled_days SMALLINT[]` (ISO 1=man..7=søn) på `workout_templates`. Settes via `TemplateMenuSheet` (⋯). Hjem henter `/api/templates` (nå med `scheduled_days`) og viser en widget + `WeekPlanSheet`-popup som grupperer maler per ukedag. Avledet klient-side — ingen nytt endepunkt.

**Tech:** FastAPI + psycopg, Supabase, Next.js 16 + TS, Vitest+RTL, pytest. Branch `feat/ukeplan` fra main (gjort).

**Referanse:** spec `docs/superpowers/specs/2026-06-14-trening-surface-redesign-design.md` (seksjon D + «Valgt arkitektur»).
